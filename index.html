Recreating the Scurvy DAGs with Beautiful Visualizations

The Scurvy Story: A Data Science Tragedy

Scurvy was a devastating disease that affected sailors on long voyages. The cure was discovered in 1747, but due to a misunderstanding about the cause, the cure was lost for over 150 years. The story involves three different understandings of the data generating process:

  1. 1747 Understanding: Lemons prevent scurvy (correct but incomplete!)
  2. Misguided Belief: Acid kills bacteria that causes scurvy (wrong!)
  3. 1928 Understanding: Vitamin C prevents scurvy (the real mechanism)

DAG 1: The 1747 Understanding (Correct but Incomplete)

Historical Context: In 1747, James Lind discovered that lemons prevent scurvy through a controlled experiment. However, the understanding was incomplete - they knew lemons worked but not why.

# Create a DAG showing the 1747 understanding: Lemons → Scurvy Prevention
import daft
import matplotlib.pyplot as plt

# Create a PGM object
pgm = daft.PGM(dpi=150, alternate_style="outer")

# Add nodes with professional styling
pgm.add_node("lemons", "Lemons", 1, 1, aspect=3, scale=1.1,
             plot_params={
                 'facecolor': 'lightyellow',
                 'edgecolor': 'darkorange',
                 'linewidth': 2,
                 'alpha': 0.8,
             })

pgm.add_node("scurvy_prevention", "Scurvy", 3.25, 1, aspect=3, scale=1.1,
             plot_params={
                 'facecolor': 'lightgreen',
                 'edgecolor': 'darkgreen',
                 'linewidth': 2,
                 'alpha': 0.8,
             })

# Add edge from Lemons to Scurvy Prevention
pgm.add_edge("lemons", "scurvy_prevention")

# Render the DAG
pgm.render()

# Add title
plt.title("Simplified Lind's Model (1747)", fontsize=14, fontweight='bold', pad=20)
Text(0.5, 1.0, "Simplified Lind's Model (1747)")

DAG 2: The Misguided Belief (Wrong Understanding)

Historical Context: Over time, people came to believe it was the acid in lemons that killed bacteria which was causing scurvy. This led to lemons being replaced by limes (cheaper but less Vitamin C) or just using acids like vinegar, causing scurvy to return.

# Create a DAG showing the misguided belief: Acid → Bacteria in Food → Scurvy
import daft
import matplotlib.pyplot as plt

# Create a PGM object
pgm = daft.PGM(dpi=150, alternate_style="outer")

# Add nodes with professional styling
# Using red/orange tones to indicate this is the wrong understanding
# Layout: diagonal from top-left to bottom-right
pgm.add_node("acid", "Acid", 1, 2, aspect=3, scale=1.1,
             plot_params={
                 'facecolor': 'lightcoral',
                 'edgecolor': 'darkred',
                 'linewidth': 2,
                 'alpha': 0.8,
             })

pgm.add_node("bacteria_food", "Bacteria\nin Food", 2.5, 1.5, aspect=3, scale=1.1,
             plot_params={
                 'facecolor': 'lightpink',
                 'edgecolor': 'crimson',
                 'linewidth': 2,
                 'alpha': 0.8,
             })

pgm.add_node("scurvy", "Scurvy", 4, 1, aspect=3, scale=1.1,
             plot_params={
                 'facecolor': 'lightgreen',
                 'edgecolor': 'darkgreen',
                 'linewidth': 2,
                 'alpha': 0.8,
             })

# Add edges from Acid to Bacteria in Food to Scurvy
pgm.add_edge("acid", "bacteria_food")
pgm.add_edge("bacteria_food", "scurvy")

# Render the DAG
pgm.render()

# Add title
plt.title("Updated Bacterial Theory (Late 19th Century)", fontsize=14, fontweight='bold', pad=20)
Text(0.5, 1.0, 'Updated Bacterial Theory (Late 19th Century)')

DAG 3: The 1928 Understanding (Complete and Correct)

Historical Context: In 1928, the true mechanism was discovered - it was Vitamin C (ascorbic acid) that prevented scurvy. This complete understanding finally explained why lemons worked and why the acid theory was wrong.

# Create a DAG showing the 1928 understanding: Vitamin C → Collagen Synthesis → Scurvy
import daft
import matplotlib.pyplot as plt

# Create a PGM object
pgm = daft.PGM(dpi=150, alternate_style="outer")

# Add nodes with professional styling
# Layout: diagonal from top-left to bottom-right
pgm.add_node("vitamin_c", "Vitamin C", 1, 2, aspect=3, scale=1.1,
             plot_params={
                 'facecolor': 'lightblue',
                 'edgecolor': 'darkblue',
                 'linewidth': 2,
                 'alpha': 0.8,
             })

pgm.add_node("collagen_synthesis", "Collagen\nSynthesis", 2.5, 1.5, aspect=3, scale=1.1,
             plot_params={
                 'facecolor': 'lightyellow',
                 'edgecolor': 'darkorange',
                 'linewidth': 2,
                 'alpha': 0.8,
             })

pgm.add_node("scurvy", "Scurvy", 4, 1, aspect=3, scale=1.1,
             plot_params={
                 'facecolor': 'lightgreen',
                 'edgecolor': 'darkgreen',
                 'linewidth': 2,
                 'alpha': 0.8,
             })

# Add edges from Vitamin C to Collagen Synthesis to Scurvy
pgm.add_edge("vitamin_c", "collagen_synthesis")
pgm.add_edge("collagen_synthesis", "scurvy")

# Render the DAG
pgm.render()

# Add title
plt.title("Simplified Modern Understanding (1930s onwards)", fontsize=14, fontweight='bold', pad=20)
Text(0.5, 1.0, 'Simplified Modern Understanding (1930s onwards)')
